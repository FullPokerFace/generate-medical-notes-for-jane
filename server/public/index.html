AI Notes for Jane

Chrome extension that auto-generates SOAP notes on janeapp.com

Download Extension (.zip)

Installation Instructions

  1. Download the zip file using the button above.
  2. Unzip the file — right-click → Extract All. You'll get a folder called chrome-extension.
  3. Open Chrome and go to chrome://extensions.
  4. Enable Developer mode using the toggle in the top-right corner.
  5. Click Load unpacked and select the chrome-extension folder you just unzipped.
  6. Navigate to any janeapp.com page — the panel will appear automatically in the top-right corner.
Note: Because this extension is not published on the Chrome Web Store, you must keep Developer mode enabled. Do not remove the unzipped folder — Chrome loads the extension directly from it.
